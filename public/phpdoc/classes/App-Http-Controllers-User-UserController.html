public changePassword(ChangePasswordRequest $request, int $id) : RedirectResponse

Validates the request data, checks the current password, and updates the password. If the user is not found or the current password is incorrect, redirects back with an error message.

Parameters
$request : ChangePasswordRequest

The validated request data.

$id : int

The ID of the user whose password is being changed.
